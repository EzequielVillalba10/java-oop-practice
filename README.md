# java-oop-practice
Java OOP practice exercises covering fundamentals, problem-solving, and clean code principles.
This repository contains Java practice exercises focused on Object-Oriented Programming (OOP) fundamentals.

The exercises are academic and cover problem-solving, clean code, and basic Java concepts.

## Topics Covered
- Variables and data types
- Conditional statements and loops
- Methods and modularization
- Arrays and basic algorithms
- Object-Oriented Programming fundamentals

## Exercises
- BMI (Body Mass Index) calculator
- Interest calculation
- Leap year finder
- Palindrome checker
- Array processing with constraints

## Technologies
- Java
- IntelliJ IDEA

## Purpose
This repository is intended to demonstrate learning progress, code organization, and Java fundamentals.
